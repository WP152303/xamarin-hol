# xamarin-hol

Xamarin 실습에 사용할 문서 및 자료를 모아둔 공간입니다. Xamarin University 과정에 있는 내용 중 일부를 발췌하고 적절하게 변경하여 작성하였습니다. Xamarin.Forms로 크로스 플랫폼 앱 개발시 꼭 필요한 내용들 위주로 튜토리얼을 작성하였으며 부족한 부분은 틈틈히 업데이트 할 예정입니다.
